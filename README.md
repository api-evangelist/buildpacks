# Cloud Native Buildpacks

Cloud Native Buildpacks (CNBs) transform application source code into OCI-compliant container images that can run on any cloud, without requiring Dockerfiles. A CNCF incubating project initiated by Pivotal and Heroku in 2018, CNBs centralize container expertise through composable buildpacks, enable layer rebasing for efficient OS updates, and generate Software Bills of Materials (SBOM).
